# adk-multiagent-lab
This repository contains a comprehensive showcase of modern AI agent architectures and orchestration patterns built using the **Google AI Development Kit (ADK)** and the Gemini models.

## Project Overview

An introduction to the ADK Basic repository as a showcase of AI agent architectures.

## Deep Dive into Agent Patterns

### 1. Single Agent Pattern (`a_single_agent`)

Basic standalone agent architecture.

---

### 2. Sequential Agent Pattern (`b1_sequential_agent`)

Agents execute tasks in a predefined sequential pipeline.

---

### 3. Parallel Agent Pattern (`b2_parallel_agent`)

Multiple agents execute tasks concurrently.

---

### 4. Loop Agent / Critique-Refinement Pattern (`b3_loop_agent`)

Agents iteratively refine outputs through feedback loops.

---

### 5. Manual Sequential Flow Pattern (`b4_manual_sequential_flow`)

Custom-defined step-by-step orchestration flow.

---

### 6. Custom Agent / Dynamic Orchestrator Pattern (`c_custom_agent`)

Dynamic orchestration and custom agent coordination logic.

---

### 7. Routing Agent Pattern (`d_routing_agent`)

Agents route requests dynamically to specialized sub-agents.

---

### 8. Agent-as-a-Tool Pattern (`e_agent_as_tool`)

Agents are exposed and consumed as callable tools.

---

### 9. Agent with Session Memory Pattern (`f_agent_with_memory`)

Agents maintain contextual memory across interactions.

---

### 10. MCP Integration Pattern (`g_agents_mcp` & `mcp_tool_box`)

Integration of agents with MCP-based tooling and workflows.

---
